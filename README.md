# Online Food Ordering System - Quick Crave Caf√©

A complete web-based food ordering system designed for cafes and restaurants. This project includes a responsive customer-facing website for browsing menus and placing orders, alongside a robust admin dashboard for managing day-to-day operations, sales, and inventory.

## Features

### User Side (Client Interface)
* **Interactive Home Page**: Features hero section, featured categories, special offers, and testimonials.
* **Menu Browsing**: Categorized menu view (Signature Coffee, Fresh Pastries, Frappes & Teas, etc.).
* **Ordering System**: Users can browse items and place orders directly through the platform.
* **Contact & About**: Dedicated pages for business information and customer inquiries.
* **Responsive Design**: Built with Bootstrap 5 to ensure compatibility across mobile, tablet, and desktop devices.

### Admin Side (Management Panel)
* **Dashboard Overview**: Real-time statistics including:
    * Total Orders & Revenue
    * Pending Orders Counter
    * Daily Sales Summary
    * Visual Charts (Revenue History, Order Status Distribution)
    * Top Selling/Popular Items
* **Point of Sale (POS)**: Interface for processing in-store orders.
* **Order Management**: Track and update order statuses (Pending, Confirmed, Preparing, Ready, Completed, Cancelled).
* **Menu Management**: Add, edit, or remove menu items and categories.
* **Customer Management**: View and manage customer details.
* **Reports**: Generate sales reports and view business analytics.
* **Secure Authentication**: Session-based login system for administrators.

## Tech Stack

* **Backend**: PHP
* **Database**: MySQL
* **Frontend**: HTML5, CSS3, JavaScript
* **Framework**: Bootstrap 5
* **Libraries**:
    * Chart.js (for data visualization)
    * SweetAlert2 (for interactive alerts)
    * Bootstrap Icons
